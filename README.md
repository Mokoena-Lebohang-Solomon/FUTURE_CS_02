📧 Phishing Email Analysis & Awareness
🧾 Overview

This project analyzes phishing emails and provides awareness guidelines to help users identify and avoid phishing attacks.

🎯 Objectives
Identify phishing indicators
Classify email risks
Explain how phishing works
Provide prevention tips
🛠️ Tools
Google Admin Toolbox Messageheader
MXToolbox
Browser DevTools
📬 Samples Analyzed
Account Verification Scam
Banking Alert Scam
Invoice Scam
Job Offer Scam
⚠️ Key Indicators
Suspicious sender
Urgent language
Fake links
Generic greetings
Requests for sensitive data
🛡️ Prevention
✅ Do
Verify sender
Hover over links
Report suspicious emails
❌ Don’t
Click unknown links
Download attachments
Share passwords or OTPs
📊 Conclusion

Phishing targets human behavior — awareness is the best defense.
